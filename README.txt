SISTEMA DE JUSTIFICATIVA DIGITAL DE HORAS

COMO RODAR:

1. Instale o Python.

2. Abra a pasta do projeto no VS Code.

3. No terminal, execute:

   pip install -r requirements.txt

4. Depois execute:

   python app.py

5. Acesse no navegador:

   http://127.0.0.1:5000

6. Para ver o painel administrativo:

   http://127.0.0.1:5000/admin


OBSERVAÇÃO:
Esta é uma versão inicial/local. Para usar na empresa pelo celular dos motoristas,
o sistema precisará ser hospedado em um servidor ou computador da rede.
